# Link and Parent Screening Policy

Linked videos and web resources are provided to support weekly biology topics. These resources are selected to be useful and age-appropriate, but parents should preview videos and links according to their family’s standards before student use.

Parents make the final decision about whether a video, website, or supplemental resource is appropriate for their student. Optional resources may be skipped, replaced, or used only after parent review.

## Short Packet Note

> Parent note: Please preview linked videos and websites before student use. Resources are recommended for course support, but parents make the final suitability decision.

## Resource Format

Each major online resource should be provided in a format that works for both printed packets and computer use.

Recommended format:

- Resource title
- Clickable link in the digital version
- Short URL when practical
- QR code for printed packets
- Required or optional label
- Parent note if the topic is sensitive or worldview-related
