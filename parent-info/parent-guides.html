The Mirror Module System

Biology Foundations is designed so that parents can confidently support their student's work without needing a biology background. The course uses a mirror module system: for every public student module, there is a corresponding private parent check guide.

The public student modules (the pages on this website) are designed for students to read and work from. They intentionally do not include answer keys, sample answers, or grading notes — those belong in the parent guide so that students can do their own thinking first.

What Each Parent Guide Includes

SectionPurpose
Week Overview A plain-language summary of what students are learning this week, written for a parent who is not a biology expert.
Assignment Overview A description of all at-home tasks, what to look for, and approximately how long each should take.
Vocabulary Check The target vocabulary for the week with brief, plain-English definitions to help parents check student work.
Answer Key / Sample Answers Sample answers for guided questions, worksheets, and sorting activities. These give parents a reference point — student answers do not need to match exactly, but should show the main idea.
Grading and Check Guide Simple guidance on what "good enough" looks like for each assignment. No rubric expertise required.
Common Misconceptions The most common student mistakes or confusions for this week, and how to address them if they come up.
Tuesday Prep Note A short note on what the Tuesday session will cover and what students should have done before arriving.

Why Guides Are Not Posted Publicly

Parent guides are distributed privately — they are not posted on this public website. This is intentional:

  • It keeps student-facing materials clean and free from answer clutter.
  • It gives students the opportunity to do their own thinking before a parent checks work.
  • It gives the co-op director a way to distribute guides only to enrolled families.

If a student page on this site mentions a printed packet, worksheet, or assignment — the parent guide for that week will explain exactly what to look for when checking it.

How to Access Parent Guides

Parent guides are distributed through the co-op director at the start of each unit. If you are an enrolled family and need access to a guide, contact the co-op director directly.

If you are a prospective family reviewing this site before enrollment, the director can share a sample parent guide at the initial family meeting.

Note for Directors: Parent guides are maintained as a separate document set. They are not part of this public GitHub Pages site. If you are reviewing this site for co-op planning purposes and would like to see the full parent guide format, please contact Justin Howard directly.

Your Role as a Parent Grader

Biology Foundations follows the standard homeschool model: parents are the primary graders unless a separate arrangement has been made with the co-op director. Parent guides are designed to make that grading as straightforward as possible — you do not need to know biology deeply to use them effectively.

The Tuesday sessions are led by Justin Howard, who can address content questions directly with students. Parents remain the academic authority in the home.
