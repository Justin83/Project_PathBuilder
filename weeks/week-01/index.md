# Week 01 — Biology, Observation, and Course Habits

## Focus

Students learn what biology studies, how to set up a biology notebook, and how to distinguish observation from inference.

## Student Tasks

- Set up biology notebook or binder.
- Read the assigned OpenStax introduction section when finalized.
- Define biology, observation, inference, hypothesis, variable, and evidence.
- Complete the observation activity sheet.

## Tuesday Activity

Observation vs. inference activity using natural objects, classroom items, or mystery samples.
