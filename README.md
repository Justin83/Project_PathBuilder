# Project PathBuilder

Public-facing course materials for **Project PathBuilder: Biology Foundations**, a guided homeschool-compatible biology course using **OpenStax Concepts of Biology** as the primary free resource.

This repository is intended to organize weekly packets, worksheets, lab sheets, QR/link resource pages, parent information, and public course materials.

## Course Model

This course is designed around a weekly guided-study rhythm. Students complete selected readings and notebook work during the week, attend a Tuesday concept check-in and activity/lab session, and then complete follow-up work at home. Parents remain the primary graders unless a separate grading arrangement is requested.

## Parent Screening Note

Linked videos and web resources are provided to support weekly biology topics. These resources are selected to be useful and age-appropriate, but parents should preview videos and links according to their family’s standards before student use. Parents make the final decision about whether a video, website, or supplemental resource is appropriate for their student. Optional resources may be skipped or replaced.
